# SOCCER EDGE ENGINE - COLOR-CODED EDGE SIGNALS & RECOMMENDATIONS

## 🎨 Visual Edge Classification System

The Soccer Edge Engine uses a **confidence-tiered color coding system** to quickly identify betting opportunities.

---

## 📊 EDGE VALUE TIERS

### **Tier 1: HIGH VALUE** 🟢 GREEN
**Edge Threshold:** ≥ +25 cents  
**Market Visual:** Bright green, glowing highlight, icon ⭐⭐⭐

**Characteristics:**
- Model significantly underpriced in market
- Clear mathematical advantage
- Worth prioritizing for action
- Recommended confidence: HIGH (95%+ conviction)

**Example:**
- Model says: Draw has 35% probability = 35 cents fair value
- Market offers: Draw at 8 cents  
- Edge: +27 cents ✅ **HIGH VALUE - BACK IMMEDIATELY**

**Rules for HIGH VALUE:**
- Only recommend if data is very confident
- Use for primary betting focus
- Alert user immediately in GUI
- Log as primary recommendation

---

### **Tier 2: VALUE** 🟡 AMBER/YELLOW
**Edge Threshold:** +8 to +25 cents  
**Market Visual:** Yellow/amber, standard highlight

**Characteristics:**
- Model has meaningful advantage
- Good opportunity but not exceptional
- Worth tracking and considering
- Recommended confidence: MEDIUM (70-85% conviction)

**Example:**
- Model says: Over 2.5 has 62% probability = 62 cents fair value
- Market offers: Over at 55 cents
- Edge: +7 cents → Actually this is SMALL EDGE
- 
- Better example:
- Model says: Over 2.5 has 62% probability = 62 cents
- Market offers: Over at 50 cents  
- Edge: +12 cents ✅ **VALUE - CONSIDER THIS PLAY**

**Rules for VALUE:**
- Build positions across multiple VALUE opportunities
- Accumulate over time for better ROI
- Monitor odds movement
- Good for secondary betting strategy

---

### **Tier 3: SMALL EDGE** 🔵 BLUE/CYAN
**Edge Threshold:** 0 to +8 cents  
**Market Visual:** Cyan/light blue, subtle highlight

**Characteristics:**
- Model has slight advantage
- Marginal mathematical edge
- Risk/reward may not justify action
- Recommended confidence: LOW-MEDIUM (50-70% conviction)

**Example:**
- Model says: Under 2.5 has 48% probability = 48 cents fair value
- Market offers: Under at 45 cents
- Edge: +3 cents ✅ **SMALL EDGE - SKIP OR ACCUMULATE MANY**

**Rules for SMALL EDGE:**
- Skip unless building portfolio with many such opportunities
- Use for accumulator betting (multiple small edges)
- Educational value only in early phase
- Archive for academic tracking

---

### **Tier 4: AVOID (FADE)** 🔴 RED
**Edge Threshold:** ≤ 0 cents  
**Market Visual:** Red, warning icon ❌

**Characteristics:**
- Model is priced worse than market
- Negative expected value
- Market is smarter than model on this one
- Should NOT bet this outcome

**Example:**
- Model says: Draw has 20% probability = 20 cents fair value
- Market offers: Draw at 45 cents
- Edge: -25 cents ❌ **AVOID - DON'T BET THIS**

**Rules for AVOID:**
- Highlight in RED prominently
- Add warning exclamation mark ⚠️
- Use as contrarian signal
- Could indicate market intelligence we're missing

## 🎯 CONFIDENCE TIERS

The system also assigns **CONFIDENCE LEVELS** based on:
1. Data quality (red cards, pressure bias accuracy)
2. Edge magnitude (larger edge = more confident)
3. Team profile reliability (known vs unknown team)
4. Time in match (earlier = more uncertain)

### **Confidence Levels**

| Level | Color | Icon | Trigger |
|-------|-------|------|---------|
| **HIGH** | 🟢 Green | ⭐⭐⭐ | Edge ≥ +25 cents + strong team data |
| **MEDIUM** | 🟡 Amber | ⭐⭐ | Edge 8-25 cents OR mixed data |
| **LOW** | 🔵 Blue | ⭐ | Edge < 8 cents OR weak data |
| **UNCERTAIN** | ⚪ Gray | ? | Conflicting signals OR unknown teams |

## 🏆 TIER DECISION LOGIC

Flow chart for assigning recommendations:

```
START: Analyze Match
  ↓
Calculate edge for each outcome (DRAW, UNDER, OVER)
  ↓
Edge ≥ +25 & HIGH confidence?
  ├─ YES → 🟢 HIGH VALUE (recommend immediately)
  └─ NO → Continue
  ↓
Edge 8-25 & MEDIUM+ confidence?
  ├─ YES → 🟡 VALUE (add to watchlist)
  └─ NO → Continue
  ↓
Edge 0-8?
  ├─ YES → 🔵 SMALL EDGE (display, don't push)
  └─ NO → Continue
  ↓
Edge ≤ 0?
  ├─ YES → 🔴 AVOID (flag, don't recommend)
  └─ NO → Continue
  ↓
END: Display all tiers to user
```

## 🎨 COLOR PSYCHOLOGY IN SPORTS GAMBLING

Why these colors?

- **GREEN 🟢:** Universal signal for "GO" / positive / opportunity
- **AMBER 🟡:** Caution signal / "proceed with care" / medium risk
- **CYAN 🔵:** Information / neutral / "just so you know"
- **RED 🔴:** STOP / danger / clear warning

This aligns with traffic lights and gaming UI conventions, making the system intuitive.
