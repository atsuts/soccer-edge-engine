# Soccer Edge Engine - Launch Guide

## Quick Start

### Main App File
- **Main launcher**: `soccer_gui.py`

### Launch Methods

#### 1. VS Code / Windsurf Play Button
- Click the Run/Play button in your editor
- Automatically launches `soccer_gui.py`

#### 2. Windows Batch Launcher
- **File**: `run.bat`
- **Usage**: Double-click the file
- **Location**: Project root folder

#### 3. Terminal Command
```bash
python soccer_gui.py
```

## Requirements
- Python installed
- Run from project folder

## Notes
- Single main launcher: `soccer_gui.py`
- No additional setup required
- Works with existing project structure
